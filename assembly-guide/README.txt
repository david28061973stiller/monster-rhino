MONSTER RHINO ASSEMBLY GUIDE
============================

Integrated location
-------------------
This directory is already prepared for placement in the root of the Monster
Rhino website. The guide will be available at:

    /assembly-guide/

The homepage integration links directly to `assembly-guide/index.html`, which
also works when the website is opened locally via `file://`.

Keep the folder structure intact. The page is self-contained and uses only
the local files included in this folder. Its logo, background, signature and
fonts match the existing Monster Rhino site.

Files
-----
- index.html  Page structure and content
- style.css   Responsive website design and print layout
- script.js   Reference-image enlargement
- assets/     Branding, fonts and the four assembly reference images

Keep this directory structure intact when committing the integration.
